================================================================================
PRERREQUISITOS PRINCIPALES
================================================================================

1. CUENTA GOOGLE CLOUD
   ✓ Cuenta creada en https://cloud.google.com
   ✓ Facturación habilitada
   ✓ $300 USD de crédito gratuito por 90 días
   ✓ Proyecto GCP creado

2. HERRAMIENTAS LOCALES
   ✓ gcloud CLI (Google Cloud SDK)
   ✓ Node.js 18+ con npm
   ✓ Git
   ✓ Docker (opcional, para testing local)

3. CREDENCIALES NECESARIAS
   ✓ JWT_SECRET (generar: openssl rand -base64 32)
   ✓ DB_PASSWORD (generar: openssl rand -base64 24)
   ✓ SMTP_HOST (Gmail: smtp.gmail.com)
   ✓ SMTP_PORT (Gmail: 587)
   ✓ SMTP_USER (tu-email@gmail.com)
   ✓ SMTP_PASS (App Password de Gmail)

4. GOOGLE CLOUD CONFIGURADO
   ✓ gcloud autenticado (gcloud auth login)
   ✓ Proyecto configurado (gcloud config set project PROJECT_ID)
   ✓ APIs habilitadas:
     - cloudbuild.googleapis.com
     - run.googleapis.com
     - containerregistry.googleapis.com
     - sqladmin.googleapis.com
     - secretmanager.googleapis.com

5. CÓDIGO PREPARADO
   ✓ Repositorio clonado
   ✓ Dependencias instaladas (npm install)
   ✓ Frontend compila sin errores
   ✓ Backend valida sin errores

================================================================================
COMANDOS DE VERIFICACIÓN RÁPIDA
================================================================================

# Verificar herramientas instaladas
gcloud --version
node --version
npm --version
git --version
docker --version

# Verificar autenticación GCP
gcloud auth list
gcloud config get-value project

# Verificar APIs habilitadas
gcloud services list --enabled

# Ejecutar verificación automática completa
cd scripts/cloud-run
./check-prerequisites.sh

================================================================================
FLUJO DE DESPLIEGUE
================================================================================

1. PRERREQUISITOS (30-45 min)
   - Crear cuenta GCP
   - Instalar herramientas
   - Generar credenciales
   - Verificar: ./check-prerequisites.sh

2. SETUP GCP (10-15 min)
   - Habilitar APIs
   - Crear service accounts
   - Configurar Cloud SQL
   - Almacenar secretos
   - Script: ./setup-gcp.sh

3. BUILD & DEPLOY (15-20 min)
   - Build backend image
   - Build frontend image
   - Deploy a Cloud Run
   - Script: ./deploy.sh

4. MIGRACIÓN DB (5-10 min)
   - Conectar a Cloud SQL
   - Ejecutar migraciones
   - Script: ./migrate-db.sh

5. VERIFICACIÓN (5 min)
   - Health checks
   - Revisar logs
   - Testing funcional

TIEMPO TOTAL: 1-1.5 horas

PRÓXIMOS PASOS
================================================================================

Una vez completados los prerrequisitos:

1. Verificar prerrequisitos:
   cd scripts/cloud-run
   ./check-prerequisites.sh

2. Si todo pasa, configurar GCP:
   ./setup-gcp.sh

3. Desplegar servicios:
   ./deploy.sh

4. Migrar base de datos:
   ./migrate-db.sh

5. Verificar despliegue:
   gcloud run services list
   curl https://BACKEND_URL/health
